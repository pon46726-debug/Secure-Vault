## 🛡️ Что это?

Secure Vault — это **локальный** менеджер паролей с военным шифрованием. Твои данные никуда не утекают, всё хранится только на твоём ПК в зашифрованном виде.

**Идеально для:**
- 🔑 Хранения паролей от Minecraft-аккаунтов (Microsoft, Mojang, Hypixel)
- 💳 Защиты банковских карт и крипто-кошельков
- 🎮 Сохранения данных от Steam, Discord, Twitch
- 📧 Почтовых ящиков и соцсетей

## ⚡ Фичи

| Фича | Описание |
|------|----------|
| 🔒 **AES-256** | Военное шифрование, такое же как у NASA |
| 🧂 **PBKDF2** | 480,000 итераций хеширования — брутфорс невозможен |
| 🎨 **Красивый UI** | Тёмная тема с неоновыми акцентами |
| ⚡ **Быстрый** | SQLite — мгновенный поиск по 1000+ паролям |
| 📋 **1 клик** | Копирование пароля в буфер |
| 🎲 **Генератор** | Создание невзламываемых паролей (12-15 символов) |
| 📂 **Категории** | Сортировка по типам (Игры, Финансы, Соцсети...) |
| ⏱️ **Автоблок** | Блокировка через 5 минут бездействия |
| 💾 **Бэкапы** | Экспорт/импорт зашифрованных баз |

## 🚀 Быстрый старт

```bash
# 1. Установи все файлы и закинь в одну папку

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Запуск

## 🔐 Первый запуск
При первом запуске программа попросит задать **мастер-пароль**. 
⚠️ Запомни его! Без него доступ к базе восстановить невозможно.
